# 🍽️ RotiReady – Smart Hotel Pre-Order & Pickup System

RotiReady is a full-stack web application developed to simplify hotel food ordering. Customers can place orders online, make advance payments, and collect their food without waiting in long queues. The system also provides an admin dashboard for efficient order management and tracking.

---

## 🚀 Features

### 👨‍🍳 Customer Features
- Online food ordering
- Advance payment system
- Order tracking
- Digital receipt generation
- Responsive user interface
- English & Telugu language support
- Mobile-friendly design

### 👨‍💼 Admin Features
- Secure admin login
- Dashboard with order management
- Update order status
- View customer orders
- QR code generation for orders
- Revenue and order monitoring

## 📌 Workflow

1. Customer visits the website.
2. Selects food items and quantity.
3. Makes an advance payment.
4. Receives an Order ID and receipt.
5. Hotel prepares the order.
6. Customer tracks the order status.
7. Admin updates the order status.
8. Customer collects the order without waiting.
